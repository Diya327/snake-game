# Snake Game

## Overview
This is a classic Snake Game built using the turtle library in Python. The player controls a snake that moves around the board, eating food to grow bigger while trying to avoid running into itself and the walls.

## Features
- Classic snake gameplay
- Points system with a score display
- Responsive design for various devices
- Simple intuitive controls

## Game Controls
- **Arrow Keys**: Control the direction of the snake.
  - Up Arrow: Move up
  - Down Arrow: Move down
  - Left Arrow: Move left
  - Right Arrow: Move right

## Technologies Used
- Python
- Turtle library
